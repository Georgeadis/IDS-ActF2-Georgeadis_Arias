# IDS-ActF2-Georgeadis_Arias

Aplicación Java con interfaz grafica para el registro y consulta de alumnos.

## Funcionalidad

- Registro de alumnos con nombre, apellidos, fecha de nacimiento y programa.
- Visualización de listado completo o filtrado por tipo de programa.
- Interfaz grafica construida con Swing.

## Estructura del proyecto

- `src/`: Código fuente Java
- `bin/`: Archivos compilados (excluidos por `.gitignore`)
- `lib/`: Librerías externas (en este repo no hay)
- `.gitignore`: Exclusión de archivos innecesarios
- `README.md`: Este archivo
